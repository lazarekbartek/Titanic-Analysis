# Titanic Survival Analysis with Feature Engineering

## Overview

This project explores the Titanic dataset with a focus on how social relationships and passenger location influenced survival. Instead of relying only on demographic variables, additional features were engineered to better understand factors affecting survival.

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NetworkX

## Project Workflow

- Data cleaning
- Missing value handling
- Exploratory Data Analysis (EDA)
- Feature engineering
- Statistical analysis
- Data visualization
- Interpretation of results

## Engineered Features

- Family Survival Rate
- Family Size
- Ticket Group Size
- Ticket Survival Rate
- Cabin Letter
- Cabin Class
- Has Cabin

## Analysis Highlights

- Investigated the impact of family and ticket groups on survival.
- Analyzed cabin location and passenger class.
- Examined interactions between gender, class, and cabin location.
- Performed statistical tests to validate the observed relationships.
